R package of CSOmap demo data

# Installation

``` r
# install.packages("devtools")
devtools::install_github("lijxug/CSOmapR.demo")
```

# Usage 

## Load demo dataset
``` r
library("CSOmapR.demo")
invisible(TPM)
invisible(LR)
invisible(labelData)
```
